# flutter-web-android-animated-rotate-string

## Description
A POC project for flutter with animation.
This is not signed.

## Tech stack
- flutter
- android

## Docker stack
- matspfeiffer/flutter:beta

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
